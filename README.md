# Arkomebel

## 📋 Նկարագրություն
Կահույքների ցուցադրում խոհանոց, ննջասենյակ, հյուրասենյակ, միջանցք, պահարաններ, մանկական, բազմոց, սեղան աթոռներ

## 🛠️ Օգտագործված տեխնոլոգիաներ
- React
- Tailwind CSS
- JavaScript (JS)
- JSX

## 🚀 Տեղակայում տեղական համակարգչում

```bash
# Clone the repository
git clone https://github.com/mhermher/Arkomebel.git

# Go to project directory
cd Arkomebel

# Install dependencies
npm install

# Start the development server
npm start
```
